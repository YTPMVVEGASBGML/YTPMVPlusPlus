What is YTPMV+?

The YTPMV++ project is an ongoing effort to automatically generate YTPMV machinimas. The current method of doing so is by using YTPMV++, a fully featured application. Contributing to the YTPMV /https://github.com/YTPMVVEGASBGML/YTPMVPlusPlus/ project can be simple, challenging, and fun. Join the YTPMV++ Hub Discord!

Community Focused

Through the efforts of my own contributions, I have brought together the community of the now defunct YTPMV++ beta as well as others to help bring to light the community-focused project. The community is what brings together the project. Whether you've found it from Vinesauce or from just lurking about, you too can help the community grow.



Description

As a passion project, I have decided to keep the longevity of the YTPMV++ project originally created by Ben Brown alive through community interaction and software updates. I was inspired to learn C# and Java through this application and I'd love to see it innovated and refined. This website doubles as a means for documentation for the YTP++ software and as a way to distribute plugins and top creations, sources, SFX, music and more.

Privacy Policy



You are not being tracked. In the future I may store Discord IDs and usernames as part of an application integration but the asset store hasn't begun production yet. Privacy is valuable, unless you decide to log into this website using Discord there will be no data stored here. The source code is available on GitHub as well.
